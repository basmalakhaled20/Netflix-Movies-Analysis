# 🎬 Netflix Data Analysis

A comprehensive Exploratory Data Analysis (EDA) project on the **Netflix Movies & TV Shows** dataset using Python. The project focuses on cleaning the data, exploring patterns, and visualizing insights to better understand Netflix's content library.

---

## Project Overview

The goal of this project is to analyze Netflix's catalog and answer important business questions such as:

- What types of content are available on Netflix?
- Which countries produce the most titles?
- How has Netflix's content grown over the years?
- What are the most common ratings?
- Which genres dominate the platform?
- What trends can be discovered from the data?

---

## Dataset

The project uses the **Netflix Movies and TV Shows Dataset**, which contains information about:

- Movie & TV Show titles
- Type (Movie / TV Show)
- Director
- Cast
- Country
- Release Year
- Rating
- Duration
- Genre
- Date Added
- Description
